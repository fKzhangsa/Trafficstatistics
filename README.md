# Trafficstatistics
统计服务器建立过的网络连接，并形成图表。方便你抓住电脑中潜在的安全隐患。
## 前言
 非常希望了解到我的电脑哪些进程建立了哪些连接，所以我写了这个工具，利用这个工具可以统计一段时间内所有建立的网络连接以及他们的连接时间图。
## 使用方式
 1.点击开始监测就不用管了，一般你可以关掉所有软件挂一晚上来采集数据。
 ![image](https://user-images.githubusercontent.com/57693945/231090372-3a88549e-082f-493b-ab42-54ff4e5a7db3.png)
 2.采集完成后选择感兴趣的时间段点击更新按钮，所有连接过的ip就会出现在下拉列表中，该列表的排序方式为连接时间越长的越靠上。
 ![image](https://user-images.githubusercontent.com/57693945/231090516-8dc3f3d5-1ec1-47ea-9946-7e9f581cbe55.png)
 3.选择绘图的时间尺度并在下拉列表中选中对应ip来查看这个ip这段时间的连接情况，以及使用该连接的程序名，其中1表示该时间内处于连接中，0表示该时间内未连接。
 ![image](https://user-images.githubusercontent.com/57693945/231090768-06dec39b-f249-44ca-b2ad-9b825941f23b.png)
 4.把时间尺度缩放到1分钟，就会能看到更详细的密密麻麻的连接情况。
 ![image](https://user-images.githubusercontent.com/57693945/231090903-de003d42-8007-4629-812e-5013272bac75.png)
